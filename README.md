
## AGROMARKET – Prototipo Funcional (TRL5)

Este repositorio corresponde al desarrollo del prototipo funcional de **AGROMARKET**, un sistema diseñado para mejorar los procesos de comercialización agrícola en Sabanalarga, Casanare, conectando productores, comercios y consumidores mediante una plataforma ágil, transparente y accesible.

### Descripción general del proyecto

AGROMARKET surge como una solución tecnológica orientada a reducir pérdidas en la comercialización de productos agrícolas, fortalecer la transparencia en precios y mejorar el acceso a canales de venta más eficientes.  
El prototipo incluye la representación visual de la arquitectura funcional, los flujos de usuario y la navegación principal del sistema, con el fin de validar la experiencia de uso y la propuesta de valor.

Dado que este proyecto **no se usara como opcion de grado**, no se desarrolló una versión programada del software. En su lugar, se elaboró un **prototipo interactivo en Figma**, correspondiente a un nivel de maduración tecnológica **TRL5**, el cual permite visualizar y simular el funcionamiento del sistema en un entorno controlado.
